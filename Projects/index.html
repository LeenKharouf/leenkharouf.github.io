Lunar Lander: reinforcement learning

Implemented a reinforcement learning agent to autonomously land a lunar module in a simulated environment. The project utilises Python, PyTorch, and libraries such as OpenAI Gym to train neural networks with techniques like experience replay and epsilon-greedy exploration.


ECG Design

3D Self-Avoiding Walk: Unity

This Unity-based project, programmed in C#, presents a 3D simulation of a self-avoiding walk styled as an interactive maze. Key features include explosions that dynamically alter the path, multiple agents navigating without collisions, and enhanced camera controls for optimal viewing angles.


Game of Life: a life in colour

Developed in C# using the Unity game engine, this project is a modified version of the classic Game of Life simulation. The canvas is segmented into areas with defined boundaries, each assigning a unique color to cells within it. This addition introduces a visual dimension to the cell behavior, enhancing the aesthetics of the simulation.
