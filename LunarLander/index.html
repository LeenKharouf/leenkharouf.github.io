LUNAR LANDER

This project uses reinforcement learning to train an agent to control a lunar lander, navigating it to safely land between two flags. The agent learns through trial and error, optimising its behaviour over time using techniques like Q-learning and deep neural networks.



Before Training

The agent takes random, chaotic actions, struggling to control the lunar module.

After Training

After training, the agent successfully lands the lunar module with precision.
